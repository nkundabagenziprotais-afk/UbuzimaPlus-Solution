# Authenticated Dashboard Frame

This Phase 0 frame defines the direction for authenticated users.

## Admin Levels

1. Ubuzima+ Admin
2. PharmaCo360 Solution Admin
3. VitaPharma Tenant Admin

## Dashboard Principles

- Role-based navigation
- Summary cards
- Pending tasks
- Module activation visibility
- AI governance visibility
- Audit and security awareness
- Responsive layout
- Tenant data separation
- No generic dashboard filler
