# Ubuzima+ Public Website Frame

The website structure follows a strong corporate-service layout inspired by modern banking and enterprise platforms, adapted for Ubuzima+ health technology.

## Sections

- Utility top bar
- Main navigation
- Hero section
- Quick action cards
- Solutions
- Audience segments
- AI Center
- Trust and security
- Platform structure
- Contact and footer

The website must not copy another brand's design, colors, text, logo, or assets.
